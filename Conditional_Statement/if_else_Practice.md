# If-Else Practice Questions (C Programming)

## 🔹 Basic Level (Easy)

### 1. Positive, Negative, or Zero
Write a program to check whether a number is positive, negative, or zero.

### 2. Even or Odd
Write a program to check whether a number is even or odd.

### 3. Greater Number
Write a program to find the greater number between two numbers.

### 4. Vowel or Consonant
Write a program to check whether a character is a vowel or consonant.

### 5. Divisible by 5
Write a program to check whether a number is divisible by 5 or not.

---

## 🔹 Medium Level

### 6. Largest Among Three Numbers
Write a program to find the largest among three numbers.

### 7. Leap Year
Write a program to check whether a given year is a leap year or not.

### 8. Divisible by 3 and 7
Write a program to check whether a number is divisible by both 3 and 7.

### 9. Grade System
Write a program to assign grades based on marks:
- 90 and above → Grade A  
- 75–89 → Grade B  
- 50–74 → Grade C  
- Below 50 → Fail  

### 10. Number Size Check
Write a program to check whether a number is:
- Less than 50 → "Small"  
- Between 50 and 100 → "Medium"  
- Greater than 100 → "Large"  

---

## 🔹 Advanced Level

### 11. Palindrome Number
Write a program to check whether a number is a palindrome.

### 12. Valid Triangle
Write a program to check whether three sides can form a valid triangle.

### 13. Type of Triangle
Write a program to determine the type of triangle:
- Equilateral  
- Isosceles  
- Scalene  

### 14. Electricity Bill
Write a program to calculate electricity bill:
- First 100 units → ₹5/unit  
- Next 100 units → ₹7/unit  
- Above 200 units → ₹10/unit  

### 15. Salary Bonus
Write a program to calculate salary bonus:
- Salary > 50000 → 20% bonus  
- Salary 30000–50000 → 10% bonus  
- Salary < 30000 → 5% bonus  

---

## 🔥 Bonus Challenges

### 16. Login System
Write a program to check login credentials:
- Username = "admin"  
- Password = "1234"  
- Print "Login Successful" or "Invalid Login"  

### 17. Even/Odd Operation
Write a program:
- If number is even → print its square  
- If odd → print its cube  

---

## 💡 Notes
- Use `if` for single condition  
- Use `if-else` for two conditions  
- Use `else if` for multiple conditions
